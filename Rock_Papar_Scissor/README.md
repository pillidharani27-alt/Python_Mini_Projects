# Rock Paper Scissors Game

This mini project is a simple Rock–Paper–Scissors game built in Python.  
The user plays against the computer, which randomly chooses rock, paper or scissors.

![image alt](https://github.com/pillidharani27-alt/Python_Mini_Projects/blob/main/Rock_Papar_Scissor/Rock%20papaer%20scissor%20code%20screenshot.png?raw=true)

## Features

- User vs computer gameplay
- Computer chooses randomly
- Shows who wins each round
- Simple and beginner-friendly Python logic
